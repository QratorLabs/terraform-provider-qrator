# qrator_client_certificate (Resource)

Manages client certificates in Qrator. Supports `upload` and `letsencrypt` certificate types.

~> **Note:** This resource does not support in-place updates. Any change to the configuration requires the certificate to be deleted and recreated.

## Example Usage

### Upload Certificate

```terraform
resource "qrator_client_certificate" "upload" {
  client_id = 12345
  type      = "upload"

  certificates = [
    {
      type = "rsa"
      cert = file("cert.pem")
      key  = file("key.pem")
    }
  ]
}
```

### Let's Encrypt Certificate

Let's Encrypt issuance typically takes **15–20 minutes**. Terraform will wait for the certificate to be issued before completing the apply.

```terraform
resource "qrator_client_certificate" "letsencrypt" {
  client_id = 12345
  type      = "letsencrypt"
  domain_id = 12345
  hostnames = ["example.com", "www.example.com"]
}
```

A maximum of **12 hostnames** is supported per Let's Encrypt certificate.

### Linking a Certificate to a Domain

A certificate by itself is just stored in the certificate storage. To activate it on a domain, link it via `qrator_domain_sni`:

```terraform
resource "qrator_client_certificate" "letsencrypt" {
  client_id = 12345
  type      = "letsencrypt"
  domain_id = 67890
  hostnames = ["example.com", "www.example.com"]
}

resource "qrator_domain_sni" "example" {
  domain_id = 67890

  links = [
    {
      host        = null
      certificate = qrator_client_certificate.letsencrypt.id
    },
    {
      host        = "example.com"
      certificate = qrator_client_certificate.letsencrypt.id
    },
  ]
}
```

## Import

Certificates can be imported using the format `client_id/certificate_id`:

```shell
terraform import qrator_client_certificate.example 12345/67890
```

Where `12345` is the client ID and `67890` is the certificate ID.

~> **Note:** After import, `request_id` will be null — the certificate request will not be cleaned up when the resource is deleted. Sensitive fields (`cert`, `key`) are also not populated after import.
