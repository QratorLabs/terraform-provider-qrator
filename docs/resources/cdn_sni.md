# qrator_cdn_sni (Resource)

Manages SNI configuration for a CDN domain in Qrator.

## Example Usage

```terraform
resource "qrator_cdn_sni" "example" {
  domain_id = 12345

  entries = [
    {
      host        = "example.com"
      certificate = qrator_client_certificate.cert.id
    },
    {
      host        = "cdn.example.com"
      certificate = qrator_client_certificate.cert.id
    }
  ]
}
```

## Import

CDN SNI configuration can be imported using the domain ID:

```shell
terraform import qrator_cdn_sni.example 12345
```
