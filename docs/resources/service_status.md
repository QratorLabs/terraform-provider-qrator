# qrator_service_status (Resource)

Manages the status of a service in Qrator.

~> **Note:** Deleting this resource resets the service status to `online`.

## Example Usage

```terraform
resource "qrator_service_status" "example" {
  service_id = 12345
  status     = "online"
}
```

### Take a service offline

```terraform
resource "qrator_service_status" "maintenance" {
  service_id = 12345
  status     = "offline"
}
```

## Import

Service status can be imported using the service ID:

```shell
terraform import qrator_service_status.example 12345
```
