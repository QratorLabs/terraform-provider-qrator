# qrator_service_ips (Resource)

Manages the upstream IP addresses for a service in Qrator.

The full list is replaced on every apply via `service_ip_set`.

~> **Note:** Deleting this resource clears the IP list (sets it to an empty array).

## Example Usage

```terraform
resource "qrator_service_ips" "example" {
  service_id = 12345

  ips = [
    "192.0.2.1",
    "198.51.100.12",
  ]
}
```

## Import

Service IPs can be imported using the service ID:

```shell
terraform import qrator_service_ips.example 12345
```

After import, run `terraform plan` to see the current state read from the API.
